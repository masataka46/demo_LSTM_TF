# demo program of LSTM using tensorflow  
This is the demo program of LSTM using tensorflow.  
It needs input sin(x) + e(noise),  and predict sin(x + dx).  


# dependency  
I confirmed operation only with..  
1)python==3.6.0    
2)tensorflow==1.8.0
